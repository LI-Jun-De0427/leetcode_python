# LeetCode #1768: Merge Strings Alternately (交替合併字串)

這是一個非常經典的字串處理入門題。透過這題，我們可以學習如何優雅地處理兩個長度不同的序列。

---

## 題目敘述
給你兩個字串 `word1` 和 `word2`。
* 從 `word1` 開始，以 **交替** 的順序合併字元。
* 如果一個字串比另一個長，請將剩餘的字元直接附加在合併後的字串末尾。
* 回傳合併後的字串。

---

## 解題思路：雙指針 (Two Pointers)

這題的核心邏輯就像是「拉拉鍊」：
1. **共同部分：** 使用一個指針 `i` 同時遍歷兩個字串，直到其中一個字串結束。
2. **剩餘部分：** 檢查誰還有剩下的字元，直接把它們「貼」在結果後面。

### 為什麼這題對初學者很重要？
* 學習處理 **邊界條件**（當兩個字串不一樣長時）。
* 學習 Python 的 **字串切片 (Slicing)** 語法。
* 了解 **List 與 String 拼接** 的效能差異。

---

## 圖解範例
輸入：`word1 = "abc"`, `word2 = "pqrst"`

| 步驟 | 目前 index (i) | 加入 word1 | 加入 word2 | 合併結果 (res) |
| :--- | :---: | :---: | :---: | :--- |
| 1 | 0 | **a** | **p** | `["a", "p"]` |
| 2 | 1 | **b** | **q** | `["a", "p", "b", "q"]` |
| 3 | 2 | **c** | **r** | `["a", "p", "b", "q", "c", "r"]` |
| **剩餘** | | (無) | **st** | `apbqcr` + `st` |
