# LeetCode #2: Add Two Numbers (兩數相加)

這題是學習 **鏈結串列 (Linked List)** 邏輯最經典的入門題。

---

## 題目敘述
給你兩個 **非空** 的鏈結串列，代表兩個非負整數。
* 數字是 **反向** 儲存的（例如：`342` 儲存為 `2 -> 4 -> 3`）。
* 每個節點包含一個數字。
* 請將這兩個數字相加，並以鏈結串列的形式回傳結果。

---

## 解題思路：模擬直式加法

這題的核心其實就是我們小學學過的 **「直式加法」**。

### 1. 為什麼要「反向」儲存？
因為加法運算必須從 **個位數** 開始。題目給的鏈結串列頭剛好就是個位數，這讓我們可以直接從頭開始往後計算，並把進位交給下一組節點。

### 2. 關鍵變數
* **Carry (進位值)：** 儲存相加後超過 10 的部分（不是 0 就是 1）。
* **Dummy Head (虛擬頭節點)：** 建立一個空的開頭，方便我們之後回傳結果，不用處理第一個節點的特殊情況。
